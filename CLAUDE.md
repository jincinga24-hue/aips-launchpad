# AIPS Launchpad

## Project
Single-page web app for AIPS club at University of Melbourne. Students submit startup ideas/MVPs, get scored by committee, approved ones go on a public board for team recruitment.

## Tech
- Single `index.html` file — HTML + CSS + JS, no framework
- localStorage for persistence
- No backend, no build step

## UI Style
- Clean Apple white/grey aesthetic
- Colors: white (#fff), light grey (#f5f5f7), border grey (#d2d2d7), text (#1d1d1f, #86868b)
- AIPS green (#2DB757) for accents only
- Rajdhani font for headings, system font (-apple-system, SF Pro) for body
- Rounded corners (12px), subtle shadows, lots of whitespace

## Key Files
- `index.html` — the entire app
- `docs/PRD.md` — product requirements
- `docs/ARCHITECTURE.md` — architecture overview
